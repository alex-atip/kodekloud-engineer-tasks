# KodeKloud Engineer Tasks

Documentation of hands-on sysadmin tasks completed as part of [KodeKloud Engineer](https://engineer.kodekloud.com/) — a hands-on simulation platform where you act as a sysadmin for a fictional company (xFusionCorp Industries / Nautilus), working on real remote servers via SSH.

This repo is part of my transition from Frontend Development into DevOps/Cloud infrastructure. Each task is documented with the commands used, the reasoning behind them, and the mistakes/lessons along the way — not just the final solution.

## Tasks

| # | Task | Topic |
|---|------|-------|
| 01 | [Create Custom Apache User](linux/task-01-create-custom-apache-user.md) | Linux User Management — `useradd` with custom UID/home directory |
| 02 | [Group Creation and User Assignment](linux/task-02-create-group-all-servers-create-users-user.md) | Linux User Management — `groupadd`, secondary groups, multi-server SSH |
